# AZIK to Romaji Converter for Karabiner-Elements

Karabiner-Elements用のComplex Modifications設定ファイルです。  
AZIKの入力を通常のローマ字入力に変換します。

## 用途

タイピングゲームなど、AZIK入力が使えない環境で通常のローマ字入力に自動変換したい場合に使用します。

## インストール

1. [Karabiner-Elements](https://karabiner-elements.pqrs.org/)をインストール
2. JSONCファイルを `~/.config/karabiner/assets/complex_modifications/` に配置
3. Karabiner-Elementsの設定画面でComplex Modificationsを有効化
